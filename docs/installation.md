# Installation

## Stable release

To install YouTube Video to PDF Summarizer, run this command in your terminal:

```sh
uv add python-boilerplate
```

Or if you prefer to use `pip`:

```sh
pip install python-boilerplate
```

## From source

The source files for YouTube Video to PDF Summarizer can be downloaded from the [Github repo](https://github.com/LosMen562/youtube_video_to_pdf_summarizer).

You can either clone the public repository:

```sh
git clone git://github.com/LosMen562/youtube_video_to_pdf_summarizer
```

Or download the [tarball](https://github.com/LosMen562/youtube_video_to_pdf_summarizer/tarball/master):

```sh
curl -OJL https://github.com/LosMen562/youtube_video_to_pdf_summarizer/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd youtube_video_to_pdf_summarizer
uv pip install .
```
